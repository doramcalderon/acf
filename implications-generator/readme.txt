------------------------------------------------------------------------------------
				implications-generator
------------------------------------------------------------------------------------

	implications-generator-full.jar  es un generador de implicaciones aleatorias.
	A continuación se indican los pasos a seguir para su ejecución:
	
	1. Debe tener instalado JAVA SE 8u65
	   
	2. Copie el archivo implications-generator-full.jar en su máquina.
	
	3. Se puede ejecutar:
		- Desde la línea de comandos ejecute
			> java -jar <directorio>/implications-generator-full.jar	
			donde <directiorio> es el directorio donde se encuentra el JAR. 
			Si se encuentra en el mismo, no es necesario indicarlo.
		- Haciendo doble click sobre implications-generator-full.jar
	   
	4. Al ejecutarlo se abrirá la ventana principal donde podrá introducir el número de nodos 
		y el número de implicaciones.
		Haciendo click sobre el botón Generar, obtendrá las implicaciones aleatorias.
		Para más ayuda, haga click en Help -> Ayuda.
