\chapter{CONCLUSIONES}

	En este proyecto se han implementado dos herramientas de utilidad para el estudio de algoritmos sobre conjuntos de implicaciones:
	
	\begin{itemize}
		\item[(1)] Generador de implicaciones aleatorios.
		\item[(2)] IS Bench, herramienta para la ejecución de algoritmos y la posterior consulta de resultados.
	\end{itemize}
	
	
	Además, se han implementado dos algoritmos de cálculo de bases en Java:  Direct Optimal Basis  y CLA cuyos pseudocódigos se presentan en los artículos  \cite{bib:ref9} y 
	\cite{bib:ref1} respectivamente.
	
	También, queda definida la API para la implementación de nuevos algoritmos que pueden ser ejecutados desde IS Bench.
