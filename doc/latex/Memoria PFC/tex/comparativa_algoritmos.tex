	\chapter{COMPARATIVA ENTRE ALGORITMOS}
	
	En este capítulo, se realiza una comparativa entre los algoritmos implementados (Direct Optimal Basis y CLA) obteniendo los
	resultados con IS Bench.
	
	El siguiente experimento está basado en conjuntos de implicaciones aleatorias. Se han generado 50 conjuntos combinando dos
	parámetros: el número de implicaciones ( de 5 a 15) y el número de atributos (de 5 a 15). 
	
	Se han desarrollado dos clases para la comparación de los dos algoritmos cuyos detalles se han explicado en el capítulo \ref{chap:algoritmosBases} 
	\textit{Algoritmos de cálculo de bases}. El experimento ha sido ejecutado en un Intel Core i5, 8GB de RAM, con Windows 7 64 bits.
	
	En la siguiente tabla se muestran para cada par número de implicaciones y número de atributos, el tamaño mínimo, el tamaño máximo
	y el tamaño medio del conjunto de implicaciones generado.
